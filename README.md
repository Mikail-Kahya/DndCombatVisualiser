# Dungeons & Dragons Combat Visualiser
C++20, Custom Engine

## Overview

Custom game engine built from scratch to explore **low-level graphics programming** with Vulkan API. Features a **standalone graphics library** that links into any project via CMake. Test demo loads Ratchet & Clank 3D model into massive open space where player can freely walk around using keyboard/XInput controllers.

**Core goal:** Learn Vulkan render pipelines while maintaining clean engine architecture that supports multiple game projects.

## Tech Stack

- **C++20** (modern features, RAII)
- **Custom engine** (Own engine)
- **CMake** (modular linking)
- **Git** (version control)
